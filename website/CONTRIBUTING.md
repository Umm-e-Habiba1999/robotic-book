# Physical AI & Humanoid Robotics Book Project Constitution

## 1. Book Overview
- Title: Physical AI & Humanoid Robotics: A Practical Course
- Goal: Teach Physical AI principles and humanoid robotics
- Technology: Docusaurus, Claude Code, Spec-Kit Plus, OpenAI Agents

## 2. Folder Structure
- `docs/` → all chapter markdown files
- `blogs/` → optional blog posts
- `docusaurus.config.js` → site config
- `package.json` → dependencies
- `specs.yaml` → defines chapters, sections, authors

## 3. Chapter Structure (as per specs.yaml)
1. Introduction to Physical AI and Robotics → `01_introduction.md`
2. Sensing, Perception, and Environment Interaction → `02_sensing_perception.md`
3. Actuation, Control, and Dynamics → `03_actuation_control.md`
4. AI Agents and Decision Making → `04_agents_decision.md`

## 4. Hackathon Rules
- Use Claude Code + Spec-Kit Plus  
- Deploy book to GitHub Pages  
- Integrate RAG Chatbot with OpenAI Agents  
- Optional bonus: Subagents, Agent Skills, Signup/Signin, Personalization, Urdu Translation

## 5. Automation Notes
- `specs.yaml` drives chapter generation  
- Chapters must follow the filenames in the specs.yaml
